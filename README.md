# 安装插件

> 在Yunzai根目录执行命令安装
``` bash 
git clone https://gitee.com/haanxuan/GT-Manual.git ./plugins/GT-Manual/
```
- 安装依赖
``` bash
pnpm install --filter=GT-Manual-plugin
```
